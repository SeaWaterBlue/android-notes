
Markdown 语法汇总
===
Markdown 简介
---

Markdown是一种轻量级标记语言。  
它由 2004年 由约翰·格鲁伯（英语：John Gruber）创建。

---

主要用法
---
- 标题(两种方式)

    使用前缀方式，多个 # 进行标识，最多支持1-6级标题：

    ```
    #               一级标题 
    ##              二级标题
    ###             三级标题
    ####            四级标题 
    #####           五级标题 
    ######          六级标题
    ```

    标题一和标题二也可以使用如下方式进行标识：

    ```
    第一级标题
    ========     (至少三个)
    第二级标题
    --------     (至少三个)
    ```
